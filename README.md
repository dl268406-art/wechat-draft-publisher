# wechat-draft-publisher

一个面向 Codex 的微信公众号草稿发布技能（Skill）。

它用于把文章源稿（Markdown / 飞书导出等）整理为适合微信公众号的内容，并完成草稿箱同步流程。

## 项目定位

这个仓库的目标是提供一套可复用的流程规范，帮助创作者把“写稿 + 配图 + 排版 + 草稿同步”串成稳定工作流。

适用场景：

- 从飞书文档或 Markdown 生成公众号草稿
- 处理本地图片并按文章顺序映射
- 在上传前做内容与素材预检
- 通过 API 同步到公众号草稿箱（不直接发布）

### 文件说明

- `SKILL.md`：技能主说明，定义触发场景、默认流程、规则与注意事项
- `agents/openai.yaml`：技能元数据，便于在工具侧展示与发现
- `references/wechat_draft_pipeline.md`：流程参考与验收清单

## 核心流程（简版）

1. 确认源稿与最终素材目录
2. 建立图片占位与本地文件映射
3. 进行标题、封面、图片顺序等预检
4. 生成公众号友好的 Markdown / HTML
5. 上传素材并创建草稿
6. 校验草稿可见性与关键信息

## 注意事项

- 浏览器登录态不等于 API 调用权限，API 仍依赖白名单与凭证配置
- 上传前务必确认封面、标题长度、图片顺序
- 默认仅创建草稿，不自动发布
